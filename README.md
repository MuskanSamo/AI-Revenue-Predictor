# AI Revenue Predictor

A Deep Learning web application that predicts the revenue of AI-generated media using TensorFlow/Keras.

## Features
- Predict AI-generated media revenue
- Deep Neural Network model
- Streamlit web interface
- Built with TensorFlow, Scikit-learn and Python

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- Scikit-learn
- Pandas
- NumPy
